# Introduction
This section covers the installation, configuration, and usage of n8n, an open-source workflow automation tool. In my cybersecurity home lab, n8n is used to automate detection and response workflows by integrating tools like Wazuh, VirusTotal, and other security or infrastructure services.  
The goal is to build low-code automation pipelines that reduce manual work and improve incident response times.
