# Introduction
This section covers the installation, configuration, and usage of Kali Linux in my cybersecurity home lab. Kali Linux provides a robust environment with preinstalled penetration testing tools, making it ideal for red team simulations, network assessments, and security research.  
The goal is to set up a clean, secure, and well-documented Kali environment that can integrate with other components of the lab.

## Requirements
- Proxmox VE
- Kali Linux ISO
- A virtualized or physical machine with at least:
  - 2+ CPU cores
  - 4 GB RAM
  - 20 GB storage
- Internet connectivity for package updates
- Familiarity with basic Linux commands

## Notes and Tips
- Update your system immediately after installation.
