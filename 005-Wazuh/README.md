# Introduction
This section covers the setup, deployment, and management of the Wazuh security platform in my cybersecurity home lab.  
The goal is to build a centralized monitoring system capable of detecting threats, collecting logs, and integrating with other security tools for incident response.

## Requirements
- Proxmox VE
- Wazuh OVA
- Minimum 4GB Storage Size

## Notes and Tips
- Be sure to set a static IP address, subnet mask, and default gateway in the Wazuh configuration file.
